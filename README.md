# Java GUI BMI Calculator

## 1. General Introduction
This project is a simple **Java GUI application** that calculates a user's Body Mass Index (BMI).  
The user enters height and weight, and the program displays the calculated BMI in the GUI.

---

## 2. Features

### Core Features
- Input fields for height and weight  
- Button to calculate BMI  
- Display area for BMI result  

### Additional Notes
- Uses a very simple Swing layout  
- Designed for beginners learning GUI programming  

---

## 3. Technologies

- **Java** — https://www.java.com  
- **JDK (Java Development Kit)** — https://adoptium.net/

### Steps
